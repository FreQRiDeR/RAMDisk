Create, unmount RAM disks on macOS!

* 100% Written is Swift
* Uses built in macOS CLI tools! Diskutil and HDIUTIL
* Input custom names, sizes
* Lists mounted RAM disks and easily unmounts them!
  (Ram disks will be completely erased!)
  
* By FreQRiDeR, and Claude.

Usage

Input custom RAMDisk name, if desired. Input size in MB, GB.
Select desired file system format. 
WARNING! UNMOUNTED RAMDISKS WILL BE FULLY ERASED!!!
BACKUP CONTENTS TO PHYSICAL DISK IF YOU WANT IT SAVED!!
